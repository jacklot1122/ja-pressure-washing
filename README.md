# JA Pressure Washing Website

Professional pressure washing services website.

## Features

- Responsive design (mobile, tablet, desktop)
- Hero section with logo and call-to-action
- Services showcase
- Time lapse video gallery
- Photo gallery
- About section
- Contact form
- Smooth scrolling navigation

## Deployment

This site is ready to deploy on Vercel:

1. Push to GitHub
2. Connect your GitHub repo to Vercel
3. Deploy!

## Local Development

To run locally, simply open `index.html` in a browser or use a local server:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`
